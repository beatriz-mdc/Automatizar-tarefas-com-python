# Automatização de tarefas com python

# Sobre o projeto
Uma empresa possui uma base de dados de produtos que precisam ser cadastrados em seu site.

# Objetivo
Otimizar o cadastro dos produtos no sistema.

- O código desenvolvido pode ser consultado em [Análise](https://github.com/beatriz-mdc/Automatizar-tarefas-com-python/blob/main/analise.md) e [Código Auxiliar](https://github.com/beatriz-mdc/Automatizar-tarefas-com-python/blob/main/codigo_auxiliar.md).
